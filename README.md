# AI Agent Interface Demo

A modern AI Agent Workspace demo built for Fiverr / Upwork portfolio presentation. It showcases an AI automation workbench with an app shell, agent library, task workflow, command panel, automation timeline, and integration dashboard.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Responsive dashboard UI
- Lucide icons

## Demo Highlights

- Dark high-end AI SaaS interface
- Sidebar navigation for Agent Library, Task Queue, Knowledge Base, Integrations, and Settings
- Agent cards for Sales, Support, Research, Content, and Data Analyst agents
- Task workflow states: input received, agent reasoning, tool execution, result generated, and human review
- Chat / command panel with prompt input, AI response area, suggested actions, and live status
- Automation timeline with trigger, processing, review, and completed states
- Integration section for Gmail, Slack, Notion, Google Sheets, CRM, and API

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Build for production:

```bash
npm run build
```

## Portfolio Positioning

This demo is designed to present AI SaaS product UI capabilities, including workflow orchestration, intelligent dashboards, automation status tracking, and business tool integrations. It can be adapted into client projects such as internal AI assistants, sales automation tools, customer support copilots, research agents, or analytics assistants.
